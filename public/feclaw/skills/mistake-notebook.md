# Skill: 错题整理

当用户需要整理/记录错题时，按以下步骤执行：

> **错题目录位置**: `/workspace/mistakes/`（AI Agent 的工作区根目录下）
> 所有错题文件均记录在此目录中，用户可随时通过 Agent 查看或复习。

## 1. 建立错题目录
- 执行: `mkdir -p /workspace/mistakes`
- 如果用户指定了科目（如数学），创建子目录: `mkdir -p /workspace/mistakes/数学`

## 2. 记录错题

### ⚠️ 信息收集原则
- **最大化利用用户提供的所有信息** — 用户发了文字、图片、语音转文字、上下文描述等，**全部都要记录到错题文件中**
- 如果用户提供了额外信息（如考试来源、老师评语、个人反思等），也一并记录
- 不要只挑自己觉得"重要的"写——用户愿意提供的内容即使看似辅助信息，也可能对后续复习有价值

每道错题创建一个独立 Markdown 文件，格式：

```markdown
# 错题记录 — {日期}_{序号}

## 题目
{题目内容 / 图片引用}

## 我的错误解答
{用户当时写的错误答案}

## 正确答案与解析
{正确答案及解析过程}

## 知识点分析
{涉及的知识点列表，如"一元二次方程求根公式"、"函数单调性判断"等}

## 错题出处
{如果用户提供了来源信息，如"高一期中考试"、"教辅第x页第x题"、"某次作业"等；不明确则留空}

## 错因分析
{知识点盲区 / 粗心 / 思路错误 / 概念混淆 / 审题不清}

## 额外记录
{用户提供的其他所有辅助信息，不遗漏}

## 复习提示
{下次复习时需要注意的点}
```

### 2a. 用户发图片
如果用户以图片形式发送题目：
1. 将图片复制到错题目录: `cp {图片VFS路径} /workspace/mistakes/{科目}/`
2. 创建对应的 .md 文件，引用该图片

### 2b. 用户发文字
如果用户以文字形式发送题目和解答：
直接创建 .md 文件，填入文字内容

## 3. 整理后
- 告知用户已记录，说明错题文件的位置
- 可提供同类练习题巩固
