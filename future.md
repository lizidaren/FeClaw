# FeClaw — 未来蓝图

> FeClaw 不只是一个智能体平台，更是一套面向 Agentic Learning 的产品生态。
> <br>Agent 是地基，而楼正在盖。

---

## Zentrim（格物所）

**一句话：** 你自己的无纸化笔记本。

- 拍照→VLM→HTML 自动结构化，随手拍进知识点
- 手写墨迹 + 文字 + 图片混排的自由画布
- 全搜索（向量+全文），找到了还能溯源回原图
- 版本历史管理，每一次修改都有记录

**状态：** 🚧 Mobile 端开发中，画布 + 后端管线已跑通

---

## Universe（学习宇宙）

**一句话：** PBL（项目式学习）的社交协作空间

让全班同学围绕一个主题（如"深圳河污染治理方案"）组建项目团队：
- 分工议题，各自调研，协作产出
- 项目有 deadline、里程碑、成果展示
- FeClaw Agent 作为助教全程陪伴——查资料、提建议、检查漏洞
- 最终产出可以是一份研究报告、一个 Web App、一个 PPT

**状态：** 📐 设计中

---

## Work（工作台）

**一句话：** 网页上的桌面级工作环境

当 Zentrim 不够用（要写长文/做表格/写代码）时，转进 Work：
- 窗口管理器 + 文件管理器（VFS 原生集成）
- VS Code Server（code-server）
- 文档/表格/演示（LibreOffice Online）
- 终端（SSH + 内建沙箱）
- 每个窗口可拉 Agent 进来协作

**状态：** 📐 功能定位已定，实现路线待定
