数据库配置

MySQL 是 FeClaw 的底层数据库。请确保已安装 MySQL 8.0+。

💡
请注意：FeClaw 会自动创建「数据库名」指定的数据库和表。

设置管理员账号

创建超级管理员账号。配置完成后用此账号登录。

🔒
密码要求：至少 8 位。建议使用字母+数字+符号的强密码。

你有哪些 API 账号？
