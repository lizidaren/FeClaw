# 长期记忆

这是 Agent 的长期记忆存储区，用于保存重要的跨会话信息。

## 记忆结构

### 核心记忆
存放最重要、需要长期保留的信息：
- 关键决策记录
- 重要的任务结果
- 用户核心偏好

### 主题记忆
按主题分类存储的记录：
- 📅 日期相关的事件记忆
- 📁 项目相关的进展记忆
- 💡 学习相关的知识记忆

### 临时记忆
短期需要但可能过期的信息：
- 近期任务上下文
- 临时参考信息

## 记忆规则

1. **价值判断**: 只有有价值的信息才写入长期记忆
2. **格式规范**: 使用清晰的标题和结构化内容
3. **定期清理**: 过期或无效的记忆会被清理
4. **优先级**: 核心记忆 > 主题记忆 > 临时记忆
